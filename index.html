WHAT IS A WEB

A web is actually interlinked documents in hypertext form that can be accessed via the internet. Know that web is the short word for WWW(World Wide Web)

KEY COMPONENTS OF THE WEB

  1. Hypertext
  2. HTTP(Hypertext Transfer Protocol)
  3. URL(Uniform Resource Locator)
  4. Web browser
  5. Web Servers

HOW THE WEB WORKS

  1. A user enters a URL in a web browser.
  2. The browser sends an HTTP request to the web server associated with the URL inputed.
  3. The web server processes the request and returns the requested web page
  4. The browser receives the response and displays it to the user if found and when not found it displays the error code 404
